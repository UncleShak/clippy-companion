# Clippy Companion 📎

Your retro coding buddy is back — and this time he's watching your code.

Clippy Companion brings the iconic Microsoft Office assistant to VS Code as a live error companion. He lives in your Explorer panel, reacts to your errors in real time, and keeps you company while you code.

![Clippy Companion in action](media/clippy/clippy_preview.png)

---

## What it does

- **Watches your code for errors** across all languages VS Code supports
- **Reacts instantly** when errors appear with Clippy's classic animations
- **Shows you exactly where** the problem is — file and line number
- **Click the bubble** to jump straight to the error
- **Celebrates with you** when you fix everything
- **Keeps you company** with idle animations when your code is clean

---

## How it works

Clippy lives in a panel inside your Explorer sidebar, just like VS Code Pets. When your language server reports an error, Clippy reacts immediately — no configuration needed.

Use **Previous** and **Next** to cycle through multiple errors. Click the speech bubble to jump directly to the problem line.

---

## Supported languages

Clippy works with any language that has a VS Code extension providing diagnostics. This includes:

- JavaScript and TypeScript (built in)
- Python (requires the Python extension)
- Java (requires the Red Hat Java extension)
- C/C++, Go, Rust, and many more

---

## Installation

1. Install the extension from the VS Code Marketplace
2. Open the Explorer panel (`Ctrl+Shift+E`)
3. Scroll down to find **Clippy Companion**
4. Start coding — Clippy will let you know if something goes wrong

---

## Tips

- Clippy only reacts to **errors**, not warnings — so he won't bother you for minor issues
- If you have multiple errors, use the **Previous / Next** buttons to browse them
- Toggle **Sound** on if you want the classic Clippy sound effects
- Clippy will play different animations depending on what's happening — watch for the `Alert`, `Thinking`, and `Congratulate` animations
